VRCOMM Product Spec Sheets
==========================

Place product specification files in this folder.
The Engineer Agent will load them automatically.

Supported file formats:
  .txt  — plain text spec sheets
  .md   — markdown spec sheets

Naming convention (use brand name, lowercase, spaces → underscores):
  hillstone_networks.txt
  sangfor.txt
  sophos.txt
  a10_networks.txt
  etc.

Each file should contain:
  - Product family overview
  - Key models and their specifications
  - Use cases
  - Technical requirements (throughput, users, features)

Files added here will be picked up automatically within 5 minutes (cache TTL).
