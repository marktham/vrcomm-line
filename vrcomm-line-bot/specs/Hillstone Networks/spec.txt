HILLSTONE NETWORKS — Product Spec Sheet
Website: https://www.hillstonenet.com/
Category: Next-Generation Firewall (NGFW), Network Security

PRODUCT FAMILIES
================

1. SG-6000 Series — Hardware NGFW / Enterprise Firewall
   - SG-6000-A1000   : 1 Gbps NGFW throughput, 512K sessions, for SMB (up to ~50-100 users)
   - SG-6000-A2000   : 2 Gbps NGFW throughput, 1M sessions, SMB-Mid (100-300 users)
   - SG-6000-A3000   : 4 Gbps NGFW throughput, 2M sessions, Mid-size enterprise (300-500 users)
   - SG-6000-A5000   : 8 Gbps NGFW throughput, 4M sessions, large branch/HQ
   - SG-6000-X1      : 20 Gbps FW throughput, data center / large enterprise
   - SG-6000-X2      : 40 Gbps FW throughput, carrier-grade
   - SG-6000-X4      : 80 Gbps FW throughput, large ISP/DC
   - SG-6000-E       : Entry-level series for small office, 500 Mbps NGFW

2. CloudEdge (CE) Series — Virtual Firewall / Cloud NGFW
   - Runs on VMware, KVM, AWS, Azure, Alibaba Cloud
   - CE-500V / CE-1000V / CE-2000V — throughput tiers: 500 Mbps to 2 Gbps virtual
   - Same feature set as hardware: URL filtering, IPS, App-ID, SSL inspection

3. iSource (Network Detection & Response — NDR)
   - AI-based threat detection and investigation platform
   - Ingests logs from Hillstone firewalls + 3rd party feeds
   - Detects lateral movement, C2 traffic, insider threats
   - Complements SIEM/SOAR deployment

4. CloudView — Centralized Management
   - SaaS-based management console for all Hillstone devices
   - Multi-tenant, supports SD-WAN policy, VPN hub management

KEY FEATURES
============
- Application identification: 3,000+ applications
- IPS signatures: 10,000+ (updated daily)
- URL categories: 120+ million URLs
- SSL/TLS deep inspection (including TLS 1.3)
- Threat Intelligence feeds (cloud-based)
- SD-WAN with QoS and link failover
- IPSec / SSL VPN (remote access + site-to-site)
- HA (Active-Active and Active-Passive)
- Zero Trust Network Access (ZTNA) module
- Multi-instance / VSYS for segmentation

SIZING GUIDE
============
- Up to 50 users         : SG-6000-E or A1000
- 50-150 users           : SG-6000-A1000 / A2000
- 150-500 users          : SG-6000-A2000 / A3000
- 500+ users (HQ/Campus) : SG-6000-A5000 or X1
- Data Center / High TP  : SG-6000-X1 / X2 / X4
- Cloud/Virtual env      : CloudEdge CE series

LICENSING
=========
- Base license: Firewall + Routing + VPN (included)
- Security subscription (annual): IPS + AV + URL Filter + App Control + Botnet + Cloud Sandbox
- Support license: 8x5 or 24x7 (HW replacement + software updates)
- Management: CloudView (SaaS) or HSM-2000 (on-prem)

COMMON USE CASES AT VRCOMM CUSTOMERS
=====================================
- Replace aging firewall at branch offices (A1000/A2000)
- HQ perimeter firewall with full NGFW inspection (A5000/X1)
- NDR deployment alongside existing SIEM (iSource)
- Cloud workload protection (CloudEdge on AWS/Azure)
